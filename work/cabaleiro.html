CABALEIRO

Cabaleiro, a mobile app dedicated to modern fashion enthusiasts, combines seamless shopping with curated collections to elevate users' style effortlessly.

This project centered around redesigning the app's account section to better align with Cabaleiro's aesthetic and functionality goals. By focusing on ease of navigation and brand alignment, the project sought to enhance the overall user experience, making it effortless for users to manage their profiles, view order history, and curate their wishlists.

Task: Redesign of the Account Section
Role: UI/UX Designer
Tools: Figma, Photoshop

THE GOAL

The primary objective of this project was to redesign the account section of Cabaleiro to create a seamless, visually cohesive, and user-friendly experience. The goal was to make it easy for users to navigate essential functions like profile settings, order history, and wishlist management. By refining the user interface, Cabaleiro aimed to strengthen its brand identity and boost user engagement in the competitive fashion industry.

THE PROCESS

The redesign process focused on understanding user needs and current industry trends to create an intuitive experience that aligned with Cabaleiro’s brand. Initial ideas were sketched out to map key features, allowing flexibility in design and ensuring that essential elements were effectively integrated. These ideas were later developed into functional wireframes, emphasizing streamlined navigation and user-friendliness.

THE RESULTS

The completed redesign resulted in a stylish, functional account section that significantly enhanced Cabaleiro’s brand image. The new layout made it easier for users to navigate profile settings, order history, and wishlists, aligning with the needs of fashion-focused users. The final product not only reflected Cabaleiro’s modern aesthetic but also boosted user satisfaction and positioned the brand competitively in the digital fashion retail space.

BRAINSTORMING

Objective: Gather ideas and explore potential layouts that meet both user needs and industry trends.

Actions: Conducted a brainstorming session to explore various layout concepts, focusing on essential features like order history, profile settings, and wishlists. Sketching out multiple design options allowed for flexibility and creativity, helping to identify the most effective elements and structures for a fashion-oriented account section.

PROTOTYPING

Objective: Translate selected ideas into low-fidelity wireframes to test functionality and user interaction.

Actions: Developed grayscale wireframes to focus on functionality without design distractions. Created an initial prototype that enabled user testing, ensuring the layout provided an intuitive experience and that each element was strategically placed for ease of use.

HI-FIDELITY WIREFRAMES

Objective: Finalize the design with brand-aligned visuals and polished UI elements.

Actions: Built high-fidelity wireframes incorporating Cabaleiro’s branding elements, including color schemes, typography, and design details that reflect its modern, fashion-forward aesthetic. This phase ensured that the interface was both visually cohesive and fully functional, with every element contributing to an elegant, effortless navigation experience.
